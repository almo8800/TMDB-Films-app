# TMDP-app :eyes:
Приложение со списком фильмов и детальной информацией о них из TMDB API.

**Технологии** :wrench: 
- UIKit
- Compositional Layout
- URLSession (написал собственный сетевой слой)
- MVVM-С
- RxSwift
- KingFisher
  
**Будущие доработки**
- Хеширование, плейсхолдеры на UI-элементах  
- Поиск по фильмам
